# 🎮 Tic Tac Toe Game

A fun and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.  
Play the classic game online with a clean UI and responsive design.

## ✨ Features

- 🎯 Classic 2 Player Tic Tac Toe Gameplay  
- ❌⭕ Turn-based X and O moves  
- 🏆 Automatic Winner Detection  
- 🤝 Draw Match Result  
- 🔄 Restart / Reset Game Button  
- 📱 Fully Responsive Design  
- 🎨 Clean and Modern UI  
- ⚡ Fast Performance using Vanilla JavaScript  
- 🖱️ Easy Click Controls  

---

## 💻 Tech Stack

- 🌐 **HTML5** – Page structure  
- 🎨 **CSS3** – Styling & layout  
- ⚙️ **JavaScript** – Game logic  
- 🚀 **GitHub Pages** – Live hosting platform
